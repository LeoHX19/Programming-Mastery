# Programming Mastery

A curated collection of programming examples across multiple languages, designed to provide clear, practical implementations of fundamental and advanced programming concepts.

## Languages

* Python
* C
* C++
* Java
* JavaScript
* Go *(Coming Soon)*
* Rust *(Coming Soon)*

## Repository Structure

Each language is organized into dedicated directories containing examples that progress from basic syntax to advanced topics.

### Topics Covered

* Basics
* Variables and Data Types
* Operators
* Conditional Statements
* Loops
* Functions
* Arrays and Collections
* Strings
* File Handling
* Object-Oriented Programming
* Algorithms
* Practice Problems
* Mini Projects

## Naming Convention

Programs follow a numbered naming format for consistency and easy navigation.

**Examples**

* `01_Hello_World.py`
* `02_Variables.py`
* `03_User_Input.py`
